## ğŸš€ Installation

### Prerequisites
- Flutter SDK (version 3.0 or higher)
- Dart SDK
- Android Studio or VS Code with Flutter extensions
- A Firebase project

### Setup Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/dayummson/nudge-beta-app.git
   cd nudge-beta-app
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Configure Firebase**
   - Create a new Firebase project at [Firebase Console](https://console.firebase.google.com/)
   - Add Android/iOS apps to your Firebase project
   - Download `google-services.json` (Android) and `GoogleService-Info.plist` (iOS)
   - Place them in the respective platform directories

4. **Run the app**
   ```bash
   flutter run
   ```

## ğŸ“– Usage

1. **Launch the App**: Open Nudge on your device
2. **Add an Expense**: Tap the "+" button to create a new expense entry
3. **Tag with Hashtags**: Use the hashtag input to categorize your expense (e.g., #food, #travel)
4. **Save & Track**: Hit save to store your expense in the cloud
5. **View History**: Browse your expense history with organized tags
6. **Edit on the Go**: Tap any chip to edit existing hashtags seamlessly
